# Pricing Recommendation - Private AI Chat Server

Offer: Private AI Chat Server on Ubuntu 24.04 LTS
Plan: Private AI Chat Server - Ubuntu 24.04 LTS
Publisher/company context: ALSOURI LLC

## Recommended license model

Usage-based monthly billed plan

## Recommended price input option

Per vCPU

## Recommended starting price

```text
USD/hour: 0.02 per vCPU
```

## Recommended VM software reservation pricing

Do not offer VM software reservation pricing for version 1.0.

## Rationale

A per-vCPU price is more scalable than a flat rate because the software charge grows with VM size. This is appropriate for AI workloads where customers may deploy larger VMs to run larger models.

For version 1.0, avoid complex per-market or per-vCPU-size pricing. Start with simple usage-based per-vCPU pricing and revise after usage data is available.

## Suggested later pricing tiers

- Standard CPU plan: 0.02 USD/hour per vCPU
- Premium/GPU-ready plan: higher rate in a separate plan
- Enterprise/support plan: separate offer or private plan

## Notes

Do not use Free for the first public listing unless the goal is purely adoption and testing. Free creates installs but does not validate commercial demand.

Flat rate is easier to understand, but it underprices large VM deployments and overprices very small test deployments. Per-vCPU is the better default for this product category.
