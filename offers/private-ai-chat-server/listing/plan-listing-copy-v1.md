## Plan summary

```text
Ready-to-use Open WebUI and Ollama AI chat server on Ubuntu 24.04 LTS, with secure local backend binding and no preloaded model.
```

## Plan description

```text
This standard plan provides a ready-to-use private AI chat server built on Ubuntu 24.04 LTS. It includes Open WebUI for browser-based chat, Ollama as the local model runtime, Docker Compose for service management, firewall-ready configuration, and a systemd service for reliable startup after reboot.

The plan is designed for developers, IT administrators, consultants, labs, and organisations that want a clean self-hosted AI chat environment on Azure without manually installing and wiring the core components together.

No LLM model is preloaded in this plan. After deployment, customers can connect to the VM and pull their preferred Ollama-compatible model based on their VM size, licensing requirements, and performance needs.

Ollama is configured to bind locally on the VM rather than being exposed publicly, while Open WebUI is made available through the web interface. This provides a practical baseline for private AI experimentation, proof-of-concept work, internal testing, and future AI workflow development.
```
